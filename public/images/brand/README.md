# Brand assets

Drop the logo here as **`logo.jpg`** (or `logo.png`).

The site auto-detects the file. If it's missing, the typographic wordmark in
`components/nav/Logo.tsx` is shown as a fallback.

Recommended:
- Square or rectangular crop, transparent background preferred for PNG
- Minimum 256×256 px (used at small sizes in the nav, larger on the loader screen)
